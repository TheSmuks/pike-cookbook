# Architecture

## Overview

The Pike Cookbook is a Docusaurus 3.9.2 documentation site that serves as both a reference manual and practical cookbook for Pike 8.0 programming.

## Structure

For detailed architecture documentation, see [docs/architecture.md](docs/architecture.md).

## Key Components

| Component | Description |
|-----------|-------------|
| **Docusaurus** | Documentation framework with MDX support |
| **Pike Examples** | Standalone `.pike` files demonstrating language features |
| **AutoDoc Highlighter** | Custom React component for Pike AutoDoc syntax |
| **Search** | Local search via `@easyops-cn/docusaurus-search-local` |
