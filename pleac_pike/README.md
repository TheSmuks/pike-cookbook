# PLEAC-Pike for Pike 8.0

Complete implementation of Perl Cookbook recipes in Pike 8.0.

## Features

- Modern Pike 8 syntax and idioms
- `#pragma strict_types` throughout
- 20 comprehensive chapters
- Production-ready examples

## Chapters

1. **Strings** - Text processing with Unicode support
2. **Numbers** - Mathematical operations and conversions
3. **Dates and Times** - Calendar and time handling
4. **Arrays** - Array manipulation and algorithms
5. **Hashes** - Mapping operations and functional patterns
6. **Pattern Matching** - Regular expressions with PCRE
7. **File Access** - File I/O operations
8. **File Contents** - Reading and writing files
9. **Directories** - Directory and filesystem operations
10. **Subroutines** - Functions and closures
11. **References and Records** - Data structures
12. **Packages and Modules** - Module system
13. **Classes and Objects** - Object-oriented programming
14. **Database Access** - SQL database connectivity
15. **User Interfaces** - Terminal and GUI programming
16. **Process Management** - Process control and IPC
17. **Sockets** - Network programming
18. **Internet Services** - HTTP, FTP, email, DNS
19. **CGI Programming** - Web application development
20. **Web Automation** - Scraping and API interaction

## Pike 8 Features Used

- Safe indexing operator (`->?`)
- Future/Promise for async operations
- Enhanced standard library modules
- Modern type system with `#pragma strict_types`
- SSL/TLS improvements
- Connection pooling

## Usage

Each chapter is a self-contained HTML file with embedded code examples.
Extract and run examples directly with Pike 8.0+.

```bash
pike example.pike
```
