Welcome to the Pike Cookbook

A comprehensive collection of practical recipes and examples for programming in Pike 8.0.

What is Pike?​

Pike is a dynamic programming language with a syntax similar to C and Java. It features:

  • High-level - Fast development with powerful abstractions
  • Fast execution - Compiled to bytecode, runs efficiently
  • Simple to learn - Clean syntax, easy to read and write
  • Type safety - Optional strict typing with #pragma strict_types
  • Built-in data types - Advanced string, array, and mapping support

This cookbook is organized into four main sections:

Foundation skills for Pike programming:

  • Strings, numbers, arrays, and hashes
  • Dates and times
  • Pattern matching
  • Subroutines and functions

File Operations​

Working with files and data:

  • File access and permissions
  • Reading and writing file contents

Network & Web​

Network programming and web automation:

  • CGI programming
  • Socket programming
  • Web automation and HTTP
  • Internet services and protocols

Advanced Topics​

Advanced Pike programming:

  • Classes and objects
  • References and records
  • Modules and packages
  • Process management
  • User interfaces

About the Examples​

Each recipe includes:

  • Working code - Copy and paste directly into your projects
  • Modern Pike 8 - Using current best practices and features
  • Practical solutions - Real-world problems solved
  • Explanations - Clear comments and documentation

Getting Started with Pike​

Install Pike and start coding:

# Check if Pike is installed
pike --version

# Run a Pike script
pike myscript.pike

# Interactive Pike shell
pike
